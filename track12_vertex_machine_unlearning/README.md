# Federated LoRA Adapters for DPDP Machine Unlearning Compliance

Welcome to the **Track 12 Vertex Machine Unlearning** repository. This project implements and validates a mathematically exact, $100\%$ compliant machine unlearning pipeline for deep learning systems under the Digital Personal Data Protection (DPDP) Act and GDPR.

---

## The Architectural Choice: Rejecting the Hessian

Traditional machine unlearning methods rely on **Influence Functions** and **Hessian Inversions** to perform post-hoc weight adjustments. We explicitly discard these methods for enterprise systems:
*   **Computational Bottleneck**: Inverting the Hessian matrix of a modern LLM is computationally intractable ($O(P^3)$ complexity on parameter count $P$).
*   **Auditing Risk**: Influence functions only approximate the gradient shift, leaving residual data traces that fail strict regulatory audits and expose the enterprise to legal liability.

Instead, we implement **Federated Cohort LoRA Isolation**. By freezing the base model and training cohort-specific low-rank adapters, we achieve $100\%$ legal data deletion through simple file shredding and memory hot-unloading, with $0.0\%$ catastrophic forgetting to the base model and other cohorts.

## Routing Architecture

```mermaid
flowchart TD
    Query[User Query + Metadata] --> Router{Dynamic Request Router}
    Router -->|Cohort 1| ApplyC1[Bind Cohort 1 Adapter]
    Router -->|Cohort 2| ApplyC2[Bind Cohort 2 Adapter]
    Router -->|Default| BaseOnly[Bypass Adapters]

    subgraph Serving Container (vLLM / Triton)
        ApplyC1 --> InferenceC1[y = x W_0^T + scaling * x A_1^T B_1^T]
        ApplyC2 --> InferenceC2[y = x W_0^T + scaling * x A_2^T B_2^T]
        BaseOnly --> InferenceBase[y = x W_0^T]
    end

    InferenceC1 --> Output[Output Response]
    InferenceC2 --> Output
    InferenceBase --> Output

    subgraph DPDP Deletion
        Trigger[Deletion Request] --> Evict[Unload Cohort 1 from memory]
        Evict --> Delete[Shred l1_A.npy / l1_B.npy files from disk]
    end
```

## How to Reproduce Telemetry

To execute the LoRA unlearning simulation and generate the compliance audit metrics locally:

1.  **Run the simulation**:
    ```bash
    python3 lora_unlearning_sim.py
    ```
    This script trains two cohort adapters on custom inputs, writes the weights to disk, hot-unloads and deletes Cohort 1's weights, and compares output vectors to verify unlearning.

2.  **Inspect the audit report**:
    ```json
    cat unlearning_audit_metrics.json
    ```
